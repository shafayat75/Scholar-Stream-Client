## Project Name: ScholarStream

*ScholarStream is a full-stack Scholarship Management Platform that helps students find, apply, and manage scholarships efficiently. Moderators and admins can review and manage applications seamlessly.

## Purpose
ScholarStream is designed to streamline the scholarship process for students while providing moderators and admins tools to manage applications, reviews, and payments securely and efficiently.

## Key Features =====>>>>>
### **Student Role
- Browse available scholarships with full details.
- Apply for scholarships online.
- Track application status (pending, processing, completed, rejected)  
- Submit reviews and feedback for completed scholarships  
- Pay application fees securely via Stripe integration  

### **Moderator Role
- View and manage all submitted applications  
- Approve, process, or reject applications  
- Provide feedback on applications  
- Manage scholarship reviews  

### **Admin Role
- Manage all users, scholarships, and applications  
- Dashboard analytics: applications count, reviews, payment status  
- Role-based access control (Student, Moderator, Admin)  
- Server-side search, filtering, and pagination  

### **General Features
- JWT authentication + Google social login.  
- Responsive UI using Tailwind CSS + DaisyUI.  
- Charts and analytics for admin dashboards. 
- Secure API requests using Axios with custom hooks.  
- Fully responsive design for mobile and desktop.

## **Technologies & NPM Packages
- React (Frontend).  
- Node.js & Express (Backend).  
- MongoDB (Database).  
- React Router – Client-side routing.  
- @tanstack/react-query – Data fetching, caching,isLoading. 
- Axios – HTTP requests.  
- SweetAlert2 – Beautiful alerts.  
- React Icons – UI icons.  
- Framer Motion – Animations.  
- React Hook Form – Form handling.  
- Stripe – Payment integration.
- React Chart – Dashboard analytics.
- Helmet Provider - For Better Title UI route rendering based.
- DaisyUI + Tailwind CSS – UI styling and creating Table.
